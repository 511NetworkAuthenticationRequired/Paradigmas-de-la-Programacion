# Consigna
Definir recursivamente el combinatorio de m tomados de a n:

$$
C_n^{\,m} = \frac{n!}{(n - m)!\ m!}
$$

## Soluciones
### Opción 1
```hs
-- product [1..n] = n!
combinatoria :: Int -> Int -> Int
combinatoria n m = div (product [1..n]) (product [1..(n - m)] * product [1..m])
```

### Opción 2
```hs
combinatoria :: Int -> Int -> Int
combinatoria n m
    | m == 0 || m == n = 1
    | otherwise = combinatoria (n-1) (m-1) + combinatoria (n-1) m
```

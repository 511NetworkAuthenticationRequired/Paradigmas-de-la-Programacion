# Consigna
Escribir un programa que devuelva el enésimo término de la serie de Fibonacci.

$$
Fib(n)=
\begin{cases}
1, & \text{si } n = 1 \text{ o } n = 2 \\
Fib(n-1) + Fib(n-2), & \text{si } n > 2
\end{cases}
$$

## Solución:
```hs
fibonacci :: Int -> Int
fibonacci 0 = 0
fibonacci x
    | x == 1 || x == 2 = 1
    | x > 2 = fibonacci (x - 1) + fibonacci (x - 2)
```

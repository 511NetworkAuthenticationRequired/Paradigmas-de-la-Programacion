# Consigna
Dada la siguiente función

```hs
fa [] = []
fa (a:[]) = [a]
fa (x:xs) = fa xs ++ "x"
```

Decir cuáles de las siguientes afirmaciones son verdaderas:
- [ ] fa "abcd" R = "abcx"
- [X] fa "abcx" R = "xxxx" 
- [ ] fa "a" R = "ax" 
- [X] fa "xxxx" R = "xxxx"
- [X] fa "" R = ""
- [ ] fa "" R = "x"
- [ ] fa "xxxy" R = "y"

> [!NOTE]
> `fa` devuelve una cadena de `x` con tantas `x` como la longitud de la lista menos 1, excepto que si la lista tiene un solo elemento, devuelve esa letra.

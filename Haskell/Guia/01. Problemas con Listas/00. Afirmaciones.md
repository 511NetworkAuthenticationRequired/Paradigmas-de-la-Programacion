# Consigna
Dada la siguiente función

```hs
fi [x] y = []
fi (x:xs) y = x : fi xs y
```

Marcar cuáles de las siguientes afirmaciones son verdaderas:
- [ ] fi "abcd" 'a'      R = "ab"
- [ ] fi "abcd" 'e'      R = "abcde"
- [ ] fi "edcb" 'a'      R = "edcb" : 'a'
- [ ] fi "abcd" 'a'      R = "aba"
- [X] fi "abcde" 'f'     R = "abc" ++ "d"
- [ ] fi "abcde" 'f'     R = "abc" ++ "d" y
- [X] fi "xyzw" 'w'      R = "xyz"

> [!NOTE]
> La función `fi` copia todos los elementos menos el último. Y el parámetro `y` no se usa.

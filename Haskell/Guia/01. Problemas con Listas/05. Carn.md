# Consigna

Sea una cadena \( s \) tal que se cumple:

$$
\text{long}(s) \ge n
$$

Se solicita implementar una función $carn(s, n)$ que retorne eln-ésimo carácter de s.

## Solución
```hs
carn :: String -> Int -> Char
carn s n
    | length s < n = error "No cumple la especificacion"
carn (x:xs) 1 = x
carn (x:xs) n = carn xs (n - 1)
```

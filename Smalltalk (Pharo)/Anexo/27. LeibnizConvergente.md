# Consigna
Dada la siguiente fórmula de Leibniz para el cálculo de $\pi$:

$$
\pi = 4 \sum_{n=0}^{\infty} \frac{(-1)^n}{2n+1}
$$

Obtener el valor de $\pi$ dado un **n** y un **ε** de control que responda a:

$$
\left| \sum \frac{(-1)^{n-1}}{2(n-1)+1} - \sum \frac{(-1)^n}{2n+1} \right| < \varepsilon
$$

## Solución:
```st
| numero epsilon serie |

numero:= (UIManager default request: 'Ingrese n.') asInteger.
epsilon:= (UIManager default request: 'Ingrese epsilon.') asNumber.

serie:= [ :n |
    (n = 0)
        ifTrue: [ 1 ]
        ifFalse: [
            | termino |
            termino:= ((1 negated) raisedTo: n) / ((2 * n) + 1).
            (termino abs < epsilon)
                ifTrue: [ 0 ]
                ifFalse: [ termino + (serie value: (n - 1)) ]
        ]
].

numero:= serie value: numero.

Transcript show: 'La serie devuelve: ', (numero printShowingDecimalPlaces: 6).

UIManager default alert: 'La aproximación de π con la serie de Leibniz es de ', ((4 * numero) printShowingDecimalPlaces: 6).
```

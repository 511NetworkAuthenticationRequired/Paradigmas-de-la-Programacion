# Consigna
Dada la siguiente fórmula de producto de Wallis para el cálculo de π:

$$
\frac{\pi}{2} = \prod_{n=1}^{\infty} \left( \frac{2n}{2n-1} \cdot \frac{2n}{2n+1} \right)
$$

Obtener el valor de π, dado un **n** y un **ε** de control, verificando que la diferencia entre los dos últimos elementos calculados sea menor que el mismo.

## Solución:
```st
| numero epsilon wallis |

numero := (UIManager default request: 'Ingrese n.') asInteger.

wallis:= [ :n |
	(n = 0)
		ifTrue: [ 1.0 ]
		ifFalse: [
			| m |
			m:= n * 2.0.
			(m / (m - 1) ) * (m / (m + 1)) * (wallis value: (n - 1))
		]
].

numero := wallis value: numero.

Transcript show: 'La productoria devuelve: ', (numero printShowingDecimalPlaces: 6).

UIManager default alert: 'La aproximación de π con la productoria de Wallis es de ', ((2 * numero) printShowingDecimalPlaces: 6).
```

> [!NOTE]
> Acá no entendí que quería que controle con epsilon, asíque hice recursión normal.

# Consgina
Dada una matriz cuadrada, de nxn ingresada por el usuario, devolver otra rotada a 90°. Dato: la matriz de entrada y salida **usa un arreglo unidimensional**. Ej:  
Matriz original:                Matriz rotada:  
 1 2 3                      7 4 1  
 4 5 6                      8 5 2  
 7 8 9                      9 6 3

 ## Solución:
```st
| n matriz posicion matriz_rotada posicion_rotada |

"Solicitar dimensión de la matriz"
n:= (UIManager default request: 'Ingrese la dimensión n') asInteger.

matriz:= Array new: n * n.
matriz_rotada:= Array new: n * n.

Transcript show: 'Tamaño de la matriz: ', n asString, 'x', n asString; cr.

1 to: n do: [ :x |
	1 to: n do: [ :y |
		posicion:= (x - 1) * n + y.
		matriz at: posicion put: ((UIManager default request: 'Ingrese un elemento para la posición (' , x asString , ', ' , y asString , ').') asInteger).
	]
].

Transcript show: 'Matriz ingresada: ', matriz asString; cr.

1 to: n do: [ :x |
	1 to: n do: [ :y |
		posicion:= (y - 1) * n + (n - x + 1). " Esta formula rota 90° la matriz original "
		matriz_rotada at: posicion put: (matriz at: ((x - 1) * n + y))
	]
].

Transcript show: 'Matriz rotada ', matriz_rotada asString; cr.
```

> [!NOTE]
> Acá use la formula `(x - 1) * n + y` para simular el funcionamiento que hace `array2D`.

# Consigna
Dado un arreglo, obtener otro eliminando el valor solicitado

## Opción 1
```st
| vectorE vectorS valor i |

vectorE:= #(1 -1 2 -2 3 -3).
vectorS:= Array new: ((vectorE size) - 1).

valor:= (UIManager default request: 'Ingrese un valor a eliminar') asInteger. " Como puse que el array sea de numeros, convierto el valor a numero tambien. "
i:= 1.

vectorE do: [ :elemento |
	(elemento = valor) ifFalse: [
		vectorS at: i put: elemento.
		i:= i + 1.
	]
].

Transcript show: vectorS.
```

## Opción 2
```st
| vectorE vectorS valor |

vectorE:= #(1 -1 2 -2 3 -3).
vectorS:= Array new: ((vectorE size) - 1).

valor:= (UIManager default request: 'Ingrese un valor a eliminar') asInteger. " Como puse que el array sea de numeros, convierto el valor a numero tambien. "

vectorS := (vectorE reject: [ :e | e = valor ]) asArray.

Transcript show: vectorS.
```

# Consigna
La efectividad de los códigos de bloque depende de la diferencia entre una palabra de código válida y otra. Cuanto mayor sea esta diferencia, menor es la posibilidad de que un código válido se transforme en otro código válido por una serie de errores.  
A esta diferencia se le llama distancia de Hamming, y se define como el número de bits que tienen que cambiarse para transformar una palabra de código válida en otra palabra de código válida.  
Si dos palabras de código difieren en una distancia d, se necesitan d errores para convertir una en la otra.  
Ejemplo: Entre los dos números binarios `01010101` y `00001111` hay una distancia de 4 bits, es decir, se necesitan cuatro errores para transformar un código en el otro.  
a) Desarrollar un programa que dado dos números binarios de 8 bits, indicar la distancia de Hamming correspondiente a los mismos.  
b) Desarrollar un programa que dado 2 lista con números binarios de 8 bits, determine la distancia de hamming total existente entre ambas.

## Soluciones:

### a)
```st
| ingreso num1 num2 distancia |

Transcript clear.

ingreso:= [
	| num |
	num:= ''.
	[(num size = 8) and: [(num allSatisfy: [ :c | '01' includes: c ])]] whileFalse: [ num:= UIManager default request: 'Ingrese un numero'. ].
	num
].

num1:= ingreso value.
num2:= ingreso value.
distancia:= 0.

Transcript show: num1, ', ', num2; cr.

num1 withIndexDo: [ :bit :i |
	((num2 at: i) = bit) ifFalse: [ 
		distancia:= distancia + 1.	
	]
].

Transcript show: 'La distancia es: ', distancia asString; cr.
```

### b)
```st
| carga lista1 lista2 distancia |

Transcript clear.

carga := [ :lista |
    | num continuar |
    continuar:= true.
    [continuar] whileTrue: [
        num:= ''.
        [(num size = 8) and: [num allSatisfy: [ :c | '01' includes: c ]]] whileFalse: [
            num:= UIManager default request: 'Ingrese un numero'.
            num ifNil: [ num := '' ].
        ].
        lista add: num.
        continuar := (UIManager default confirm: '¿Desea agregar otro numero?').
    ].
    lista
].

lista1:= carga value: (OrderedCollection new).
lista2:= carga value: (OrderedCollection new).
distancia:= 0.

Transcript show: 'Lista 1: ', lista1 printString; cr.
Transcript show: 'Lista 2: ', lista2 printString; cr.

((lista1 size) = (lista2 size)) 
	ifFalse: [ UIManager default alert: 'Ajo y agua, no cargaste listas de igual tamaño.' ]
	ifTrue: [
		lista1 withIndexDo: [ :codigo :i |
			codigo withIndexDo: [ :digito :j |
				(((lista2 at: i) at: j) = digito) ifFalse: [ distancia:= distancia + 1 ]
			]
		] 
	].

Transcript show: 'La distancia es: ', distancia asString; cr.
```

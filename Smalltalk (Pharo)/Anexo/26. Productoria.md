# Consigna
Hacer la productoria:

$$\prod_{i=1}^{1000} (2x + 3)^i$$

Detenerse cuando **n = 1000** o bien cuando:

$$\left| (2x + 3)^n - (2x + 3)^{n-1} \right| < e$$

donde **x** y **e** son ingresados por el usuario.

## Solución:

```st
| xx e productoria salida log10N exponente mantisa |

xx:= (UIManager default request: 'Ingrese x (entero).') asInteger.
e:= (UIManager default request: 'Ingrese e (decimal).') asNumber.

productoria:= [ :x :i |
	| n |
	n := ((2 * x) + 3) raisedTo: i.
	log10N:= n log: 10.
	exponente:= log10N floor.
	mantisa:= 10 raisedTo: (log10N - exponente).
	Transcript
		show: 'i= ', i printString;
		show: ' | (2x+3)^i ≈ ', (mantisa printShowingDecimalPlaces: 5);
		show: 'e', exponente printString;
		show: ' | Δ= ', diff printString;
		cr.
	
	((i = 1000) or: [ (((n) - (((2 * x) + 3) raisedTo: (i - 1))) abs) < e ])
		ifTrue: [ n ]
		ifFalse: [ n * (productoria value: x value: (i + 1)) ]
].

salida:= productoria value: xx value: 1.

log10N:= salida log: 10.
exponente:= log10N floor.
mantisa:= 10 raisedTo: (log10N - exponente).

Transcript
	cr;
	show: 'Resultado (notación científica): ';
	show: (mantisa printShowingDecimalPlaces: 5);
	show: 'e';
	show: exponente printString;
	cr.
```

# Consigna

Dado un texto, interpretar las señales de escape y decodificarlo. La señal de escape està dada por `~`, seguido del caracter y ocurrencias del mismo. EJ: `'Hola M~u2ndo'='Hola Muundo'` y hacer el proceso inverso

## Expansión
```st
| sec sec_s n c i |

sec:= '¡Qu~i4ero caf~e2, Bu~r7pla!'.
sec_s:= ''.
i:= 1.

[ i <= sec size ] whileTrue: [
	((sec at: i) = $~)
		ifTrue: [ 
			n:= (sec at: i + 2) digitValue.
			c:= sec at: i + 1.
			sec_s:= sec_s, (String new: n withAll: c) asString.
			i:= i + 3.
		]
		ifFalse: [
			sec_s:= sec_s, (sec at: i) asString.
			i:= i + 1
		]
].

UIManager default alert: sec_s.
```

## Compresión
```st
| sec sec_s n c i j |

sec := '¡Quiiiiero cafee, Burrrrrrrpla!'.
sec_s := ''.
i := 1.

[ i <= sec size ] whileTrue: [
    j := 1.
    [ (i + j <= sec size) and: [ (sec at: i) = (sec at: i + j) ] ] whileTrue: [
        j := j + 1
    ].

    (j > 1)
        ifTrue: [ sec_s := sec_s , '~', (sec at: i) asString , j asString ]
        ifFalse: [ sec_s := sec_s , (sec at: i) asString ].

    i := i + j.
].

UIManager default alert: sec_s.
```

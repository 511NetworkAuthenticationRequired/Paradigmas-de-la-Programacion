# Consigna
Calcular el número **e**, teniendo en cuenta el siguiente límite:

$$
e = \lim_{n \to \infty} \left(1 + \frac{1}{n}\right)^n
$$

**Épsilon (ε):** se debe definir un valor entero positivo que controle la diferencia entre un término y el anterior.  
El proceso debe **terminar cuando esa diferencia sea despreciable (ínfima)**.

**n:** cantidad de veces a iterar.  
Se utiliza como control en caso de que la condición anterior no se cumpla (es decir, un límite máximo de iteraciones).

## Solución:
```st
| numero epsilon anterior actual flag |

numero:= (UIManager default request: 'Ingrese n.') asInteger.
epsilon:= (UIManager default request: 'Ingrese epsilon.') asNumber.
anterior:= 0.
flag:= false.

1 to: numero do: [ :n |
	actual:= ((1 + (1 / n)) raisedTo: n).
	((n > 1) and: [((anterior - actual) abs) < epsilon])
	"(((anterior - actual) abs) < epsilon)"
		ifTrue: [ 
			UIManager default alert: 'e ≈ ', (actual printShowingDecimalPlaces: 6).
			flag:= true.
			^actual.
		]
		ifFalse: [
			anterior:= actual.
		]
].

flag ifFalse: [ UIManager default alert: 'No se alcanzó el criterio de ε. Último valor ≈ ', (actual printShowingDecimalPlaces: 6) ]
```

# Consigna
Distancia de Levenshtein. La **distancia de Levenshtein**, **distancia de edición** o **distancia entre palabras** es el número mínimo de operaciones requeridas para transformar una cadena de caracteres en otra.

Por ejemplo, la distancia de Levenshtein entre "casa" y "calle" es de 3 porque se necesitan al menos tres ediciones elementales para cambiar uno en el otro.

1. casa → cala (sustitución de 's' por 'l')  
2. cala → calla (inserción de 'l' entre 'l' y 'a')  
3. calla → calle (sustitución de 'a' por 'e')

a) Desarrollar un programa que dado dos palabras devuelva la longitud de levenshtein entre ellas.  
b) Desarrollar un programa que dado dos frases, devuelva la longitud de levenshtein en total entre ellas.

---
> [!NOTE]
> Al principio había entendido que había que buscar las diferencias entre dos palabras, pero no es así como funciona Levenshtein.  
>  
> La distancia se define como:
> 
> $$
> M =
> \begin{cases}
> d(i-1,j) + 1,\\
> d(i,j-1) + 1,\\
> d(i-1,j-1) + \text{costo.}
> \end{cases}
> $$
> 
> $$
> d(i,j) =
> \begin{cases}
> 0, & \text{si } i = 0 \text{ y } j = 0,\\
> j, & \text{si } i = 0 \text{ y } j > 0,\\
> i, & \text{si } j = 0 \text{ y } i > 0,\\
> \min(M), & \text{en otro caso.}
> \end{cases}
> $$
> 
> $$
> \text{costo} =
> \begin{cases}
> 0, & \text{si } a_i = b_j,\\
> 1, & \text{si } a_i \neq b_j.
> \end{cases}
> $$

## Solucion:
# a y b)
```st
| palabra1 palabra2 matriz m n posicion costo a b c minimo |

palabra1:= 'casa' asUppercase.
palabra2:= 'calle' asUppercase.

m:= palabra1 size + 1. " filas "
n:= palabra2 size + 1. " columnas "

matriz:= Array new: (m * n).

"matriz at: 1 put: 0."

1 to: m do: [ :x |
	posicion:= (x - 1) * n + 1.
	matriz at: posicion put: (x - 1)
].

1 to: n do: [ :y |
	posicion:= y.
	matriz at: posicion put: (y - 1)
].

2 to: m do: [ :x |
	2 to: n do: [ :y |
		posicion:= (x - 1) * n + y.
		((palabra1 at: (x -1)) = (palabra2 at: (y - 1))) 
			ifFalse: [ costo:= 1 ] 
			ifTrue: [ costo:= 0 ].
		a:= (matriz at: (((x - 2) * n) + y)) + 1. " eliminacion "
		b:= (matriz at: (((x - 1) * n) + (y - 1))) + 1. " inserscion "
		c:= (matriz at: (((x - 2) * n) + (y - 1))) + costo. " sustitucion "
		minimo:= (a min: b) min: c.
		matriz at: posicion put: minimo.
	]
].

1 to: m do: [ :x |
	1 to: n do: [ :y |
		| valor |
		valor := matriz at: ((x - 1) * n + y).
		Transcript show: valor printString; tab.
	].
	Transcript cr.
].

Transcript cr; show: 'Distancia de Levenshtein = ', ((matriz last) printString).
```

> [!NOTE]
> Como la entrada es un string, pueden mandarse tanto frases como palabras.

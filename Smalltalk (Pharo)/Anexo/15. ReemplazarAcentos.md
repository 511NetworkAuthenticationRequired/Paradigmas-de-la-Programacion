# Consigna
Limpiar los caracteres con acentos en una frase y devolverla sin ellos. Ejemplo: `'hólá mÚndÓ'` -> `'hola mUndO'`

## Opcion 1: parseo + codePoint
```st
| frase vocales |

frase:= 'hólá mÚndÓ'.
vocales := #($é $í $ó $É $Í $Ó) asSet.

Transcript show: frase; cr.

frase withIndexDo: [ :caracter :i |
	(caracter = $á) ifTrue: [frase at: i put: $a].
	(caracter = $Á) ifTrue: [frase at: i put: $A].
	(caracter = $Ú) ifTrue: [frase at: i put: $U].
	(caracter = $ú) ifTrue: [frase at: i put: $u].
	(vocales includes: caracter) ifTrue: [
		frase at: i put: (((caracter codePoint) - 132) asCharacter)
	]
]. 

Transcript show: frase.
```

## Opcion 2: con diccionario
```st
| frase convertir |

frase:= 'hólá mÚndÓ'.

convertir:= Dictionary new. { 
	$á -> $a.
	$é -> $e.
	$í -> $i.
	$ó -> $o.
	$ú -> $u.
} do: [ :c |
	convertir at: c key put: c value.
	convertir at: (c key) asUppercase put: (c value) asUppercase.
].

Transcript show: frase; cr.

frase withIndexDo: [ :caracter :i |
	(convertir includesKey: caracter)
		ifTrue: [ frase at: i put: (convertir at: caracter) ]
]. 

Transcript show: frase.
```

# MENSAJES DE PALABRA CLAVE

## a.  
```smalltalk
'Esto es una prueba' at: 3
```  

**Devuelve:** `'t'`  

El `at:` devuelve el elemento que se encuentra en la posición especificada (índice basado en 1).  

---

## b.  
```smalltalk
'Hola' includes: $o
```  

**Devuelve:** `true`  

El `includes:` devuelve un booleano indicando si el objeto (en este caso el carácter `$o`) se encuentra dentro de la colección (la string `'Hola'`).  

---

## c.  
```smalltalk
'hola' at: 1 put: $H
```  

**Devuelve:** `$H`  

El `at:put:` modifica el objeto en la posición indicada, pero **retorna el valor insertado**, no la colección resultante. La string queda transformada en `'Hola'`.  

---

## d.  
```smalltalk
'Paradigmas de Programación' copyFrom: 4 to: 9
```  

**Devuelve:** `'adigma'`  

El `copyFrom:to:` devuelve una subcadena (o subsecuencia en general) desde el índice inicial hasta el índice final, **inclusive**.  

---

## e.  
```smalltalk
#( 9 8 7 6 5 ) at: 3
```  

**Devuelve:** `7`  

El `at:` sobre un array devuelve el elemento ubicado en esa posición.  

---

## f.  
```smalltalk
#( 1 ( 2 3 ) 4 5 ) includes: #( 2 3 )
```  

**Devuelve:** `true`  

El `includes:` busca si existe un elemento idéntico dentro del array. En este caso, el array contiene como elemento otro array `#(2 3)`, por lo que la comparación da verdadero.  

---

## g.  
```smalltalk
#( 9 8 7 6 5 ) copyFrom: 1 to: 2
```  

**Devuelve:** `#(9 8)`  

El `copyFrom:to:` en un array devuelve una copia con los elementos comprendidos entre los índices dados.  

---

## h.  
```smalltalk
Array new: 10
```  

**Devuelve:** `#(nil nil nil nil nil nil nil nil nil nil)`  

El `Array new:` crea un nuevo array con la cantidad de posiciones indicada, inicializadas con `nil`.

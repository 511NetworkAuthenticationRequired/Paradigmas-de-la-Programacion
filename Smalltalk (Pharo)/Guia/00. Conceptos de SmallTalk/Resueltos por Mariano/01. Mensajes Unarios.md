# MENSAJES UNARIOS

## a.
```smalltalk
#( 'arreglo' 'de' 'strings' ) size
```
**Imprime:** 3

Como ya se vio, en Pharo, la notación `#( ... )` define un array. Esto significa que se crea un arreglo con los elementos que se escriben entre paréntesis y separados por un espacio. En este contexto, `size` devuelve la cantidad de elementos del arreglo. En este caso, los elementos son tres strings: `arreglo`, `de` y `strings`.

## b.
```smalltalk
'Hoy es Jueves' asUppercase
```
**Imprime:** `'HOY ES JUEVES'`

Devuelve todo el string en mayúsculas, a exepción de aquellos que no son susceptibles a esto (caracteres especiales como `!`, `#`, etc.).  

Un ejemplo mejor es:
```smallTalk
'123 !$& ú æ ç ö ø' asUppercase
```
**Imprime:** `123 !$& Ú Æ Ç Ö Ø'`

> [!CAUTION]
> El mensaje que daba la gúía no funcionaba porque decía `UpperCase` (`C` mayúscula) y no `Uppercase`

## c.
```smalltalk
'hola aquí estoy' reversed 
```
**Imprime:** `'yotse íuqa aloh'`

Ejemplo paso a paso:
1. String original: `hola aquí estoy`
2. Se toma el último caracter `y` y pasa a ser el primero.
3. Luego `o`, después `t`, y así sucesivamente hasta llegar al primero `h`.
4. Devuelve el reverso.

Cuando se envía el mensaje `reversed`, Devuelve el reverso de cualquier string, es decir, lo que hace es invertir el orden de los caracteres que forman la cadena.

> [!NOTE]
> - Si la cadena contiene espacios o acentos, se invierten igual, respetando la codificación de caracteres.
> - Si el string es un solo caracter, no hay nada que invertir: devuelve ese mismo caracter. Ejemplo: `a` reversed → `a`.
> - La operación no modifica el string original, sino que devuelve una nueva cadena invertida.

## d.
```Smalltalk
#( 4 'cinco' 6 7 ) reversed 
```

**Imprime:** `#(7 6 'cinco' 4)`

El mensaje `reversed` aplicado a un array devuelve un nuevo array con los mismos elementos, pero en orden inverso.

## e. y f.
```Smalltalk
$A asciiValue
```

**Imprime:** `65`

El prefijo `$` indica que se trata de un **caracter**. El mensaje `asciiValue` devuelve el código ASCII decimal asociado al caracter, es como el `Ord()` en pascal o `ord()` en python.

El caso inverso, para pasar de número ASCII a caracter es:
```Smalltalk
65 asCharacter
```
 **Imprime:** `$A`

 Se puede usar también `Character value: 65` y devuelve exactamente lo mismo. Este es como el `Chr()` de pascal o el `chr()` de python.

## g.
```Smalltalk
'cuál es la longitud de esta oración?' size
```

**Imprime:** 36

El mensaje `size` como ya vimos, cuenta la cantidad de caracteres de un string, sin distinción.

# CÓDIGO

ST tiene una ventana llamada “Inspector” que permite ver y cambiar las variables de instancia de un objeto. Evalúa este código y di que observas.
```Smalltalk
| a |
a := #( 1 2 sam 'joe' ( 4 5 ) ).
a at: 2 put: 3 / 4.
a inspect
```

## Explicación paso a paso

1. `| a |`  
   - Declara una **variable temporal** `a`.

2. `a := #(1 2 sam 'joe' (4 5))`  
   - Asigna a `a` un **arreglo literal** de 5 elementos:  
     1. `1` → SmallInteger  
     2. `2` → SmallInteger  
     3. `sam` → símbolo[^1] 
     4. `'joe'` → string  
     5. `(4 5)` → **array anidado** (otro arreglo de dos enteros)  

3. `a at: 2 put: 3 / 4`  
   - Reemplaza el **segundo elemento** del arreglo (`2`) por el resultado de `3 / 4`.  
   - `3 / 4` devuelve un **Fraction** (`3/4`), que reemplaza al `2`.

4. `a inspect`  
   - Abre la **ventana Inspector**, mostrando el arreglo actualizado:  
     ```Smalltalk
     #(1 3/4 sam 'joe' #(4 5))
     ```  
   - Permite **ver y modificar cada elemento** del arreglo, incluyendo el Fraction, el símbolo, el string y el array anidado.

[^1]: un símbolo como `#sam` es un nombre fijo que nunca cambia y sirve para identificar cosas en el programa. No guarda un valor ni se puede modificar, solo funciona como una **etiqueta única**. Esto es útil para diferenciar objetos similares o marcar elementos de manera que el sistema los reconozca rápidamente, porque cada vez que escribís `#sam` apunta al mismo objeto en memoria.

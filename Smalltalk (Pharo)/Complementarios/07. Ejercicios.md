Calcular la serie de: 

$$
z^{2 + j} \over (k + j!)
$$

Para $n$ términos. Con $k$ y $j$ constantes.
Se debe elaborar el algoritmo de factorial y potencia.

```st
| z j k factorial potencia numerador denominador |

z:= (UIManager default request: 'Ingrese z') asInteger.

j:= 5.
k:= 7.

factorial:= [ :n |
	(n = 0)
		ifTrue: [ 1 ]
		ifFalse: [ n * (factorial value: (n - 1)) ]
].

potencia:= [ :b :e |
	(e = 1)
		ifTrue: [ b ]
		ifFalse: [ b * (potencia value: b value: (e - 1)) ]
].

numerador:= potencia value: z value: (2 + j).
denominador:= k + (factorial value: j).

UIManager default alert: ((numerador / denominador) printShowingDecimalPlaces: 4), ' ≈ ',(numerador // denominador) printString.

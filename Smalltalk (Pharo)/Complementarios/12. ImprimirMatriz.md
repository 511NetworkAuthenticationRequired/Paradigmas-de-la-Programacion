Se tiene un archivo de texto de N líneas, donde cada fila representa una matriz. Realizar las implementaciones necesarias para imprimir cada matriz, en el formato fila x columna, similar al ejemplo de abajo.

|  12  |   6  |
|:----:|:----:|
| 1/8  | 7/2  |
|  -7  |  -6  |
|  0   |  50  |

También se debe calcular el determinante de las matrices 2 x 2 y mostrarlo en pantalla.

Ejemplo archivo de entrada.
```txt
4,2 12 6 1/8 7/2 -7 -6 0 50
2,2 5 1/5 2/7 3
1,3 -1 20 0
```

```st
| archivo lineas salto aux parser n m matriz i fila |

archivo:= 'F:\Usuario\Desktop\Paradigmas\Archivos txt\matrices.txt' asFileReference.
lineas:= (archivo contents) findTokens: { Character lf }.

parser:= [ :p |
	| auxp |
	(p includes: $/) 
		ifTrue: [ 
			auxp:= p findTokens: '/'.
			(auxp first asInteger) / (auxp last asInteger).
		]
		ifFalse: [ p asNumber ]
].


lineas do: [ :linea |
	| determinante temp |
	aux:= linea findTokens: Character separators.
	
	"Transcript show: aux; cr."
	
	temp:= (aux first) findTokens: ','.
	n:= (temp first) asInteger.
	m:= (temp second) asInteger.
	
	"Transcript show: n asString, '/', m asString; cr."
	
	aux:= aux allButFirst collect: [ :cd | parser value: cd ].
	Transcript show: 'Esperados: ', (n * m) printString, ' - Obtenidos: ', aux size printString; cr.
	
	"Calculo de determinante para matrices 2x2"
	((n = m) and: [m = 2]) ifTrue: [ 
		determinante := ((aux first) * (aux fourth)) - ((aux second) * (aux third)).
		Transcript show: 'El determinante de la matriz es ', (determinante printShowingDecimalPlaces: 2), '.'; cr.
	].

	"Impresion de matrices (SIN usar Array2D)"
	matriz:= Array new: n * m.
	i:= 0.
	
	Transcript show: 'Matriz ', n printString, 'x', m printString; cr.
	1 to: n do: [ :y | 
		fila:= '| '.
		1 to: m do: [ :x |
			i:= i + 1.
			fila:= fila, (aux at: i) asString, ' | '.
		].
		Transcript show: fila asString; cr.
	].
	Transcript show: (String new: fila size withAll: $-); cr.
]
```

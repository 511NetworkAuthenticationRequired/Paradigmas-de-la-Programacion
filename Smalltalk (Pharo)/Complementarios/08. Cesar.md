Cifrado por desplazamiento de n caracteres. Ingresar una frase y cifrarla siguiendo la regla de desplazamiento César. Utilizar como valor de desplazamiento el día de la semana remanente. 
Ejemplo: Dom= 1, Lun=2, …. Remanente: 7 – Día. Para hoy: 7 – 4.

```st
| frase offset vector salida posicion flag permitidos |

permitidos := Character alphabet , Character alphabet asUppercase , ' ¡!¿?'.

flag:= false.
[ flag not ] whileTrue: [
    frase := UIManager default request: 'Ingrese una frase (o Cancelar para salir)'.
    frase ifNil: [ ^UIManager default inform: 'Operación cancelada.' ].
    (frase isEmpty or: [ frase anySatisfy: [ :ch | (permitidos includes: ch) not ] ])
        ifTrue: [
            UIManager default alert: 'Debe ingresar una frase válido. Intente nuevamente.'
        ]
        ifFalse: [
            flag := true
        ].
].

Transcript show: frase; cr; flush.
offset:= 7 - (Date today dayOfWeek).
Transcript show: offset; cr; flush.

vector:= Character alphabet, Character alphabet . "No comtempla la ñ ni caracteres que no sean del alfabeto"
salida:= ''.

frase do: [ :letra |
	(vector includes: letra)
		ifTrue: [ 
			posicion:= vector indexOf: letra. 
			salida:= salida, ((vector at: (offset + posicion)) asString)
		]
		ifFalse: [ salida:= salida, (letra asString) ]
].

UIManager default alert: salida.
```

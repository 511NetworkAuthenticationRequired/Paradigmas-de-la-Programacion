Solicitar una frase e informar cuántas palabras comienzan y terminan con la misma letra.

```st
| frase contador |

frase:= UIManager default request: 'Ingrese una frase'.
frase:= frase asLowercase.
frase:= frase findTokens: Character separators, ',.!?;:'.

contador:= 0.

Transcript show: frase.

frase do: [ :palabra | 
	(palabra first) = (palabra last)
		ifTrue: [ contador:= contador + 1 ]
].

UIManager default alert: contador printString, ' palabras empiezan con la misma letra.'
```

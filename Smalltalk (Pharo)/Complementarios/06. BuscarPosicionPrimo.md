Busque un elemento en un arreglo donde cada elemento tiene números en forma de string (ej. ‘25’ ‘23’ … ) y si el mismo es primo, devuelva su posición. Utilizar un bloque con dos parámetros de entrada.

```st
| vector elemento busqueda |

vector:= #('25' '23' '7' '33' '0' '26' '101' '55' '1').
elemento:= ((UIManager default request: 'Ingrese el elemento a buscar') asInteger) asString.


busqueda:= [ :v :c | 
	v withIndexDo: [ :e :i |
		((e = c) and: [ (c asInteger) isPrime ]) ifTrue: [ 
			^i
		]
	]
].

(elemento isNil) ifFalse: [ 
	busqueda value: vector value: elemento.
	Transcript show: busqueda asString.
].
```
